# Yogesh76352
# Influencer Directory SaaS Platform

🚀 A SaaS-style platform built with **Django + React**, featuring analytics, paywall integration, and professional UI.  
🎓 MCA Project Submission | 📊 Power BI dashboards included

## ✨ Features
- Influencer directory with search & filter
- Analytics dashboard (Power BI integration)
- Paywall for premium access
- Modern UI with SCSS & Monaco Editor

## 🛠 Tech Stack
- **Backend:** Python, Django, SQLite
- **Frontend:** React, SCSS, Monaco Editor
- **Analytics:** Power BI
- **Deployment:** GitHub Actions
